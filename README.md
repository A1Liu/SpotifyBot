# Spotify Bot

## Plan

1. Basic features
  * Login
  * Mute audio in driver
  * Play/pause current song
  * wait for current song to finish
2. Higher-order features
  * Create and add to playlists
  * manage playlists
3. Multi-Threading
  * Bot-Handler
  * Bot Abstract class
